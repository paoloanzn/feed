### Main Topic: LLM Code Quality

**Main Thesis:** Despite claims of improvement, data shows that Large Language Models have not actually improved their ability to produce mergeable, high-quality code over the last year.

**Simple Summary:** Studies comparing how well AI code passes tests versus how mergeable the code is show that AI performance is much worse when judged by real-world quality. When looking at the actual rate of successful code merges, the data does not support the idea that AI programming skills have improved since early 2025. Statistical analysis confirms that the rate of mergeable code has been flat, meaning the perceived progress in AI coding ability is not backed up by the results.

**The Bottom Line:** > The gap between what people claim about AI coding progress and the actual performance of the models is much larger than previously thought.
